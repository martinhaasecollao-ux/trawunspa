# TrabajUN

Prototipo web (celular y escritorio) que conecta **clientes** con **técnicos** por oficio: frecuencia del servicio, referencia de precio y oferta del cliente.

Abrir `index.html` en el navegador o servir la carpeta con cualquier servidor estático. Con HTTPS, el `manifest.json` permite instalar la app en el teléfono.
